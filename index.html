Тестовое задание

  1. Создать сетку каталога состоящую из 4х колонок и 2 рядов с помощью flex (Раздел Flex-каталог с flex-direction: row)
  2. Поменять направление осей с дефолтного состояния (Раздел Flex-каталог с flex-direction: column)
  3. Выравнять один элемент по поперечной оси (Раздел Flex-каталог с flex-direction: column - 1 элемент)
  4. Задать выравнивание ряду flex-end (Раздел Flex-каталог с flex-direction: column)
  5. Создать сетку каталога состоящую из 4х колонок и 2 рядов с помощью inline-block (Раздел Inline-block)
  6. Создать сетку каталога состоящую из 4х колонок и 2 рядов с помощью float (Раздел Float)
  7. Создать html таблицу из двух рядов и 4 ячеек (Раздел HTML Table)
  8. Создать таблицу из двух рядов и 4 ячеек используя для разметки только div (Раздел CSS Table)
  9. Сделать обыкновенный аккордеон. Не важно JS или JQUERY, но обязательно с нуля(Все предыдцщие задания спрятаны в выпадающие блоки аккордеона)
